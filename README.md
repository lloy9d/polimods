# polimods

Репозиторій для зберігання та автоматичного збирання модпаків для сервера **Politime 1.21.1 NeoForge**.  
Автоматично завантажує моди за посиланнями та пакує їх у zip-архіви.  
Переглянути, звідки завантажуються моди, можна у [modlist.txt](https://github.com/lloy9d/polimods/blob/main/modlist.txt) файлі - більшість з Modrinth, деякі з CurseForge. Все відкрито і прозоро.

---

На вибір є два паки, для гри достатньо завантажити лише один:

**🔗 Poli Lite**  >  пак для слабших ПК. Майже відсутні моди на естетику = гра буде стабільною, але не такою мальовничою.

**⭐ Poli Fusion**  >  пак для сильніших ПК. Містить усі моди з Poli Lite + додаткові моди для візуальної ейфорії.

Завантажити бажаний пак можна кількома способами:

- Пряме посилання у нашому офіційному [дискорд сервері](https://discord.ptime.pp.ua) в каналі `⭐┃моди`.
- Вкладка "[Releases](https://github.com/lloy9d/polimods/releases)" на Github у цьому репозиторії.
