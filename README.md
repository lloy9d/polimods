# polimods
Репозиторій для зберігання та автоматичного збирання модпаків для сервера Politime 1.21.1 NeoForge. Автоматично завантажує моди за посиланнями та пакує їх у zip-архіви. Переглянути, звідки завантажуються моди, можна у .txt файлах - більшість з Modrinth, деякі з CurseForge. Все відкрито і прозоро
